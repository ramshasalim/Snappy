# Snappy 📸

Snappy is a simple multiplayer-style real-world challenge game built using HTML, CSS, and JavaScript. Players get random tasks based on difficulty level and upload proof images to score points.

This is the initial version of the project and will be improved with more features and better multiplayer functionality in future updates.

---

## Features
- Easy, Medium & Hard modes
- Random task generation
- Image upload system
- Leaderboard
- Modern responsive UI

---

## Technologies Used
- HTML
- CSS
- JavaScript
